# Community Impact Tracker

A modern NGO dashboard built with Python and Streamlit to help organizations monitor donations, beneficiaries, volunteers, and project performance.

## Features

- Donation Tracking
- Beneficiary Management
- Volunteer Monitoring
- Project Analytics
- Interactive Charts
- Data Tables

## Technologies Used

- Python
- Streamlit
- Pandas
- Plotly
